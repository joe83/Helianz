# Helianz QRIS Mirror Kotlin

Native Android replacement for the Flutter QRIS mirror viewer.

## What It Does

- Accepts the `Phone Link` generated by the Helianz desktop QRIS dialog.
- Lets the phone connect either by pasting the link or by scanning the new `Connect QR` shown in the cashier QRIS window.
- Polls the local cashier session JSON over the LAN.
- Displays the QR image on the phone.
- Shows live payment status, amount, order ID, and expiry.
- Keeps the final paid or terminal state visible when the session settles.

## Open In Android Studio

Open the folder below as an Android Studio project:

`d:/Project/Dental/Helianz/helianz_qris_mirror_kotlin`

## Connect Flow

1. Start a QRIS payment in the Helianz desktop app.
2. In the QRIS dialog, use the `Phone Link` directly or scan the new `Connect QR` with the Android app.
3. Keep both devices on the same local network.

## Notes

- The project allows cleartext HTTP because the cashier session is served on the local network.
- Networking still uses `HttpURLConnection` and QR images still use `BitmapFactory`.
- QR scanning is handled through ZXing embedded on Android.
