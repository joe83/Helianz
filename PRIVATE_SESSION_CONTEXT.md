## Implementation Notes

- `patient.SSN` is currently the NIK placeholder. Do NOT use SSN as a permanent home — it conflicts with US SSN usage for any international patient. Use a separate `Nik` column.
- SatuSehat staging URL: `https://api-satusehat-stg.dto.kemkes.go.id` · Production: `https://api-satusehat.kemkes.go.id`
- IHS ID = the internal patient/practitioner ID assigned by Kemkes FHIR server. Must be obtained before pushing Encounter/Condition/Procedure.
- `MAX_RETRY_COUNT = 5` — after 5 failures a record is excluded from the queue (but not deleted).
- Procedure code system for Indonesia: `http://sys-ids.kemkes.go.id/tindakan-medis` (Kemkes proprietary) or ICD-9-CM CM for procedures.
- ICD-10 system URI for SatuSehat: `http://hl7.org/fhir/sid/icd-10`
